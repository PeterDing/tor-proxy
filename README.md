# TOR Proxy

## Install docker and `dperson/torproxy`

```
./install.sh
```

## Deploy torproxy

```
./deploy.sh start_port_num end_port_num
```

## Stop running torproxy containers

```
./stop.sh
```

## Start torproxy containers

```
./start.sh
```

## Stop and remove torproxy containers

```
./destroy.sh
```
